# LP-PTUT_Scudo
Projet Tutoré de License Pro - Scudo (Flux vidéo)

## Lancer le projet
**Pensez à bien avoir deux terminal ouvert**, un pour le front et un pour le back.
(NPM => Front / PHP => Back)

### Installation des library
  
  ```bash
  npm install
  ```

  ```bash
  composer install
  ```

### Lancement du projet

  ```bash
  npm run dev
  ```

  ```bash
  php artisan serve
  ```
